# GlassBox

**A clean, powerful desktop UI for managing virtual machines.**

GlassBox is a modern VM manager built with **Tauri + React**, focused on making virtualization simple for beginners while staying powerful for developers.

---

## ✨ Goals

* 🧠 Simple enough for beginners
* ⚡ Powerful enough for developers
* 🎨 Beautiful, modern UI
* ⏱ Save time managing VMs

---

## 🚀 Features (Current)

* View all VirtualBox VMs
* Start / Stop VMs
* Clean card-based UI
* Search + filter VMs
* Desktop app via Tauri

---

## 🛠 Tech Stack

* **Frontend:** React + TypeScript
* **Backend:** Tauri (Rust)
* **VM Control:** VirtualBox CLI

---

## 📦 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/jlikescomputers556/GlassBox.git
cd GlassBox
```

---

### 2. Install dependencies

```bash
pnpm install
```

---

### 3. Run the app

```bash
pnpm tauri dev
```

---

## ⚠️ Requirements

* Node.js
* pnpm
* Rust (for Tauri)
* VirtualBox installed
